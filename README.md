# MFCard

[![License](http://img.shields.io/badge/license-MIT-blue.svg)](http://opensource.org/licenses/MIT)

Easily integrate Credit Card payments in iOS App. Swift 3.0 


### Credit Card Supported
* Visa
* Master Card 
* Discover
* JCB
* Dinners Club
